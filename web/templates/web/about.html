About the Consolidated Screening List API

What is the Consolidated Screening List?

The Consolidated Screening List (CSL) is a list of parties for which the United States Government maintains restrictions on certain exports, reexports, or transfers of items.

Overview

The Consolidated Screening List API consolidates export screening lists of the Departments of Commerce, State, and the Treasury into a single searchable database. It contains the names of restricted and denied parties with whom U.S. companies and individuals are prohibited or restricted from doing business.

Source Lists

The CSL includes data from the following lists:

Department of Commerce
  • Denied Persons List
  • Unverified List
  • Entity List
  • Military End User (MEU) List
Department of State
  • AECA Debarred List
  • Nonproliferation Sanctions
Department of the Treasury
  • Specially Designated Nationals (SDN) List
  • Foreign Sanctions Evaders List
  • Sectoral Sanctions Identifications List
  • Palestinian Legislative Council List
  • CAPTA List
  • Non-SDN Menu-Based Sanctions List

About This Application

This application provides a user-friendly interface for searching the Consolidated Screening List and viewing detailed information about entities. It also offers a RESTful API for programmatic access to the CSL data.

Key features include:

  • Search entities by name, source list, and country
  • View detailed entity information including addresses and identification documents
  • Query the live Trade.gov CSL API directly
  • Local PostgreSQL database for storing and retrieving entity data
  • RESTful API for integration with other applications

API Documentation

The REST API of this application allows you to programmatically access the CSL data. The following endpoints are available:

Endpoint Description Methods
/api/v1/entities/ List all entities or create a new entity GET, POST
/api/v1/entities/{id}/ Retrieve, update, or delete a specific entity GET, PUT, PATCH, DELETE
/api/v1/entities/search/ Search entities in the local database GET
/api/v1/entities/external_search/ Search entities using the external CSL API GET
/api/v1/entities/source_lists/ Get a list of all source lists GET
/api/v1/addresses/ List all addresses GET
/api/v1/addresses/countries/ Get a list of all countries GET
/api/v1/search-history/ View search history GET
